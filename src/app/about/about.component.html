Postgres DPA analyzer

It's a tool for comparing db table states.

Step 1. Choose table to analyze and select analyze options:

  • Find table schema changes.
  • Find changed, added or deleted rows.

Step 2. Click Gather Data button. It will collect table schema data and row changes information and save under 'analysisId'.

Step 3. Process DPA upgrade. Table schema will change, rows will be added, deleted or changed.

Step 4. Then click Analyze button, db table analysis information will be shown.
